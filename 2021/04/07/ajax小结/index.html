XMLHttpRequest实例对象的方法和属性详解

我们都知道ajax的是基于XMLHttpRequest的实例对象来实现的，所以在学习XMLHttpRequest的方法之前，我们先来写一个简单的ajax请求

有的XMLHttpRequest的方法在上面没有用到，我们先看在上面的代码中用到的方法

  1. 先看第一行代码： var xhr = new XMLHttpRequest()，代表的意思是，创建一个XMLHttpRequest的实例对象，它的实例对象可以向服务器发送异步的请求，从服务器获取到数据之后，将获取的数据渲染到页面上，不会影响页面中的其他内容，也就是进行页面的局部刷新，ajax实现的关键就是创建XMLHttpRequest的实例对象

  2. 第二行中的var xhr.open(‘get’, ‘http://localhost:3000/ajax', true)表示的意思按照设定的请求方式将请求发送到目标地址，其中，第一个参数表示的意思是请求方式，第二个参数表示请求的地址，第三参数表示发送的请求是同步还是异步，第一和第二个参数是字符串，第三参数是布尔值，其中第三个参数设置为true代表发送异步请求，false代表同步请求，默认情况下是true，即发送异步请求

  3. xhr.send()： 向服务器端传送的参数，一般get请求这里面设置为null，因为get请求的参数是放在url请求地址中进行传递的，如果是post请求的话就需要把你想传递的参数放在send()方法里，通过send()方法传送给服务器

  4. xhr.onreadystatechange ：是一个事件，当readysate发生改变的时候，就会触发这个事件

  5. xhr.readystate ：表示XMLHttpRequest的状态码，readystate一共有五个值，分别是0,1,2,3,4

    readystate的值 代表的含义
    readystate == 0 请求未初始化
    readystate == 1 请求已发送，已与服务器建立连接
    readystate == 2 send已发送，请求已接收
    readystate == 3 请求处理中
    readystate == 4 请求已完成，可以使用服务器响应的数据

  6. xhr.status ：表示的是HTTP状态码 有1xx,2xx,3xx,4xx,5xx

    xhr.status的值 代表的含义
    readystate == 1xx 请求已被接收，正在处理
    readystate == 2xx 请求已成功处理完成
    readystate == 301 将一个页面永久重定向到其他页面
    readystate == 302 将一个请求临时重定向到另一个页面
    readystate == 304 命中协商缓存的时候，服务器告诉浏览器资源未发生改变可以到浏览器的本地缓存中去获取资源
    readystate == 4xx 客户端出现错误
    readystate == 500 服务器出现错误
    readystate == 504 网关超时

  7. xhr.responseText：表示返回字符串格式的数据

  8. xhr.responseXML：表示返回XML格式的数据

  9. xhr.setRequestHeader() ：设置请求头，当我们需要制定我们发送的数据的类型的时候就会使用到这个方法，比如当我们想要传送的数据时json格式的时候我们可以使用这个方法将Content-Type设置为application/json

  10. xhr.getResponseHeader：获取响应数据的响应头，同样的，我们可以通过响应头中的Content-Type字段获取服务器端返回的数据格式
